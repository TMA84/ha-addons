# Training App - Home Assistant Add-on

Fitness training planner and tracker with AI-powered coaching.

## Features

- 📅 Weekly training plan with day-by-day view
- 🏋️ Exercise library with detailed descriptions
- ⏱️ Workout timer with speech announcements
- 📊 Body metrics tracking with charts
- 🤖 AI Coach (OpenAI) for training analysis and plan generation
- 📋 Import training plans from ChatGPT via free text
- 🔄 Multi-device sync
- 🌐 German & English

## Configuration

| Option | Description |
|--------|-------------|
| `openai_api_key` | Optional: OpenAI API key for AI features. Can also be set in the web UI per user. |
| `openai_model` | AI model (default: `gpt-4o-mini`) |

## Usage

After installation, open the web UI on port 3001. Create an account and start planning your training.
